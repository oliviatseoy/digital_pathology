# CVAT REST API

## Documents

- <http://localhost:8080/api/docs/>

## Project List

- List project (e.g., Project ID=1) <http://localhost:8080/api/projects/1>

## Task List

- List task (e.g., Task ID=1): <http://localhost:8080/api/tasks/1>

## Job List

- List job (e.g., Job ID=262): <http://localhost:8080/api/jobs/262>
- Get completed jobs: <http://localhost:8080/api/jobs?state=completed>
- Get in-progress jobs: <http://localhost:8080/api/jobs?state=in+progress>

# Label list

- List label (e.g., Project ID=1): <http://localhost:8080/api/labels?project_id=1>
